## 快速上手

1. 点击插件市场，搜索 `发布工具`，安装插件。
2. 启用插件，然后在思源笔记窗口左上角工具栏有一个飞机按钮，点击，即可使用。
   ![](./docs/images/publisher-icon.png) 
3. 文档菜单（可选），参考 [帮助文档](https://siyuan.wiki/s/20230810132040-nn4q7vs) FAQ第12条，打开文档菜单，即可使用。
