# Milestone 0 Smoke Checklist

## 1. 顶栏入口

- 关闭 `useV2UI` 后，点击顶栏发布按钮，仍显示旧版菜单
- 开启 `useV2UI` 后，点击顶栏发布按钮，显示 V2 最小面板
- 开启 `useV2UI` 后，通过设置入口打开时，仍显示同一个 V2 DOM Host，而不是 iframe 设置页

## 2. 配置来源

- 修改偏好设置中的 `useV2UI`
- 重启插件或刷新环境后，顶栏入口行为与配置一致
- 顶栏菜单开关行为不再依赖直接 `window.localStorage` 读取

## 3. 回退链路

- 人为制造 V2 初始化异常时，顶栏入口自动回退到旧版菜单
- 回退时有错误提示或日志

## 4. 回归检查

- 文档菜单开关不受影响
- 旧版发布菜单各项仍可打开
- 旧版单篇、批量、图床菜单不受影响
- 当 `useV2UI=false` 时，旧版设置页仍能通过 iframe 正常打开
